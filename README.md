## 발표 내용

### 📐 고전검사이론

시험의 **총점**을 기반으로 문항을 분석하는 이론입니다.

|개념       |핵심 질문                      |
|---------|---------------------------|
|문항변별도 (r)|이 문항을 맞힌 학생들은 전체 시험도 잘 봤을까?|
|문항추측도    |찍어서 맞힐 확률이 얼마인가?           |
|오답지 매력도  |오답지가 적절히 유혹하고 있는가?         |
|문항반응분포   |각 답지에 학생들이 골고루 반응하고 있는가?   |

**한계** — 총점만 보기 때문에 집단·검사도구에 따라 결과가 달라지는 왜곡이 발생합니다.

-----

### 🔬 문항반응이론

고전검사이론의 한계를 극복하기 위해 등장한 이론입니다.
총점이 아닌 **문항 하나하나의 고유한 특성(문항특성곡선, ICC)** 으로 분석합니다.

- **문항 특성의 불변성** — 어떤 집단이 풀어도 문항 난이도는 변하지 않습니다.
- **피험자 능력의 불변성** — 어떤 시험지를 풀어도 학생의 실력 지표는 변하지 않습니다.
- **기본가정** — 일차원성 가정, 지역독립성 가정
